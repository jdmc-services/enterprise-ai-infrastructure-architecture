# Demo Lab: Secure Enterprise RAG Assistant

**Section:** 07 — Demo Lab
**Version:** 1.0

---

## Purpose

This demo lab provides synthetic examples, sample prompts, test cases, and reference data that demonstrate how the Secure Enterprise RAG Assistant works in practice — without using any real enterprise data, systems, or configurations.

The demo lab is designed for:
- Portfolio demonstration to hiring managers and technical evaluators
- Architecture review conversations with enterprise teams
- Interview preparation — concrete examples to reference
- Onboarding new team members to the RAG system design

## Important Notice

> All content in this demo lab is **synthetic and fictional**. No real enterprise policies, network configurations, system names, employee names, IP addresses, or confidential information are included. This content exists solely for portfolio and demonstration purposes.

## How to Use This Lab

### For Portfolio Review
Read `sample-prompts.md` to see how the RAG assistant responds to realistic enterprise queries. Read `test-cases.md` to understand the security and quality validation approach.

### For Architecture Conversations
Reference `sample-enterprise-rag-app.md` as a concrete deployment example when discussing the architecture with technical evaluators.

### For Security Discussions
Reference `test-cases.md` — specifically the prompt injection and RBAC bypass test cases — when discussing the security posture of the AI system.
